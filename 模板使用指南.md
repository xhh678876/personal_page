# 🎨 模板生成器使用指南

## 快速使用

### 方法 1：命令行（推荐）

```bash
python template_generator.py your_data.json
```

或者直接传入 JSON：
```bash
python template_generator.py < xhh.html
```

### 方法 2：在 Python 中使用

```python
import json
from template_generator import generate_all_themes

# 读取 JSON 数据
with open('xhh.html', 'r') as f:
    data = json.load(f)

# 生成所有主题
generate_all_themes(data)
```

## 📋 可用主题

### 1. 紫色渐变科技风 (`gradient_purple`)
- **风格**：现代、动感、科技
- **特点**：
  - 紫色渐变背景
  - 玻璃态卡片
  - 流畅动画效果
  - 悬停交互
- **适合**：计算机、AI、工程类研究者

### 2. 暗黑极简风 (`dark_minimal`)
- **风格**：酷炫、极客、霓虹
- **特点**：
  - 黑色背景 + 绿色霓虹
  - 终端风格
  - 赛博朋克美学
- **适合**：黑客、开发者、系统研究者

### 3. 轻简学术风 (`academic_light`)
- **风格**：传统、专业、简洁
- **特点**：
  - 白色背景
  - 衬线字体
  - 经典学术布局
- **适合**：传统学科、教授、正式场合

## 🔧 自定义主题

你可以在 `template_generator.py` 中添加新主题：

```python
def template_my_custom(data):
    """你的自定义主题"""
    return f'''<!DOCTYPE html>
    <html>
    <head>
        <style>
            /* 你的 CSS */
        </style>
    </head>
    <body>
        <!-- 你的 HTML -->
    </body>
    </html>'''

# 在 themes 字典中添加
themes = {
    'my_custom': ('我的主题', template_my_custom),
    # ... 其他主题
}
```
