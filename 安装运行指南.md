# 🚀 安装与运行指南

## 方法一：使用启动脚本（最简单）

1. **确保已安装 Node.js**
   - 如果还没安装，请访问：https://nodejs.org/
   - 下载并安装 LTS 版本（推荐 18.x 或更高）

2. **双击运行 `启动.bat`**
   - 首次运行会自动安装依赖（需要几分钟）
  - 之后会自动启动服务器

3. **打开浏览器**
   - 访问 http://localhost:3000

---

## 方法二：手动命令行安装

### 前置要求

确保已安装：
- **Node.js** 18.x 或更高版本
- **npm** 或 **pnpm** 或 **yarn**

### 安装步骤

1. **打开终端/命令提示符**
   - 按 `Win + R`，输入 `cmd`，回车
   - 或者在项目文件夹按住 Shift + 右键，选择"在此处打开 PowerShell 窗口"

2. **进入项目目录**
   ```bash
   cd d:\project
   ```

3. **安装依赖**
   
   选择以下任一方式：

   **使用 npm（推荐）：**
   ```bash
   npm install
   ```

   **或使用国内镜像（如果上面很慢）：**
   ```bash
   npm install --registry=https://registry.npmmirror.com
   ```

   **或使用 pnpm（更快）：**
   ```bash
   pnpm install
   ```

   **或使用 yarn：**
   ```bash
   yarn install
   ```

4. **启动开发服务器**
   ```bash
   npm run dev
   ```
   
   或
   ```bash
   pnpm dev
   ```
   
   或
   ```bash
   yarn dev
   ```

5. **打开浏览器**
   - 访问 http://localhost:3000

## 如果遇到问题

### 问题 1: "找不到 npm 命令"

**解决方案：**
1. 重新安装 Node.js：https://nodejs.org/
2. 安装时勾选"Add to PATH"选项
3. 重启电脑
4. 打开新的终端窗口

### 问题 2: 安装速度很慢

**解决方案：**
使用国内镜像源
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 问题 3: 端口 3000 被占用

**解决方案：**
修改端口运行
```bash
npm run dev -- -p 3001
```

### 问题 4: 依赖安装失败

**解决方案：**
1. 删除 `node_modules` 文件夹
2. 删除 `package-lock.json` 文件
3. 重新运行 `npm install`

## 使用指南

### 1. 获取 API Key

#### 选项 A: Google Gemini（推荐 - 免费）

1. 访问：https://aistudio.google.com/app/apikey
2. 登录 Google 账号
3. 点击「Create API Key」
4. 复制密钥（格式：`AIzaSy...`）

#### 选项 B: OpenAI GPT-4o

1. 访问：https://platform.openai.com/api-keys
2. 创建账号并添加付款方式
3. 创建新 API Key
4. 复制密钥（格式：`sk-proj-...`）

### 2. 上传简历

1. 在网页上选择 AI 提供商（Gemini / OpenAI）
2. 输入 API Key
3. 点击上传 PDF 文件
4. 等待解析（10-30 秒）
5. 查看生成的学术主页！

### 3. 支持的简历格式

✅ 任何 PDF 格式的学术简历  
✅ 单列、双列、多列布局  
✅ 中英文混合  
✅ 建议页数 < 10 页
