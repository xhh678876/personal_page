# 📦 桌面应用使用说明

## 🎯 三种使用方式

### 方式一：网页版（原始方式）
双击 `启动.bat` → 浏览器访问 http://localhost:3000

### 方式二：桌面版（开发模式）
双击 `运行桌面版.bat` → 独立窗口打开

### 方式三：独立软件（打包分发）
双击 `打包成软件.bat` → 生成 `dist\学术主页生成器 Setup.exe`

---

## 🚀 快速开始

### 第一次使用

1. **确保已安装 Node.js**
   - 下载：https://nodejs.org/（选择 LTS 版本）
   - 安装时勾选 "Add to PATH"

2. **选择使用方式**
   - 想在浏览器用？→ 双击 `启动.bat`
   - 想要桌面应用？→ 双击 `运行桌面版.bat`
   - 想打包分享？→ 双击 `打包成软件.bat`

3. **获取 API Key**
   - Gemini（免费）：https://aistudio.google.com/app/apikey
   - OpenAI：https://platform.openai.com/api-keys

4. **上传简历，开始使用！**

## 💡 推荐使用流程

### 自己使用
1. 双击 `运行桌面版.bat`
2. 等待窗口打开（首次需要 1-2 分钟）
3. 使用应用

### 分享给他人
1. 双击 `打包成软件.bat`
2. 等待打包完成（5-10 分钟）
3. 分享 `dist\学术主页生成器 Setup.exe`
4. 对方双击安装即可使用（无需 Node.js）

## 🔧 常见问题

**Q: 双击 .bat 文件一闪而过？**
A: 右键 → 以管理员身份运行

**Q: 提示找不到 npm？**
A: 请先安装 Node.js

**Q: 桌面版打不开？**
A: 等待 3-5 秒，Next.js 需要启动时间

**Q: 打包太慢？**
A: 第一次打包需要下载 Electron，约 5-10 分钟，后续会快很多
