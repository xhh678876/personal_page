# 高級版使用指南

## 🚀 快速開始

### 運行應用
```bash
cd d:\project
python gradio_advanced_app.py
```

應用會啟動在 `http://localhost:7860` 並生成公網鏈接。

### 1. 上傳簡歷 PDF
點擊「上傳簡歷 PDF」按鈕，選擇你的學術簡歷。

### 2. 選擇 AI 提供商

點擊「✨ 生成主頁項目」，等待約 10-30 秒。

### 4. 查看結果

- 在瀏覽器中直接預覽主頁
- 部分特效（如粒子背景）會在下載後完整顯示

#### 📦 下載項目標籤
- 下載包含完整項目的 ZIP 文件
- 包含：`index.html`、`README.md`

#### 📋 數據 JSON標籤
- 查看 AI 提取的原始數據
- 可用於調試或進一步處理

## 🚀 部署到 GitHub Pages

### 方法 1：Web 界面（推薦）

1. **下載項目 ZIP**
   - 從 Gradio 應用下載 `homepage_project.zip`
   - 解壓到本地

2. **創建 GitHub 倉庫**
   - 訪問 https://github.com/new
   - 倉庫名：`你的用戶名.github.io`（個人主頁）
     或任意名稱（項目主頁）

3. **上傳文件**
   - 點擊「上傳文件」
   - 拖拽解壓後的所有文件
   - 提交

4. **啟用 GitHub Pages**
   - Settings → Pages
   - Source: `main` branch
   - 保存

5. **訪問你的主頁**
   - 個人主頁：`https://你的用戶名.github.io`
   - 項目主頁：`https://你的用戶名.github.io/倉庫名`

### 方法 2：Git 命令行

```bash
# 解壓 ZIP
unzip homepage_project.zip
cd homepage_project

# 初始化 Git
git init
git add .
git commit -m "Initial commit: Academic Homepage"

# 連接遠程倉庫
git remote add origin https://github.com/你的用戶名/倉庫名.git

# 推送
git branch -M main
git push -u origin main
```

然後在 GitHub 設置中啟用 Pages。

## ❓ 常見問題

**Q: Gradio 預覽和下載後的效果一樣嗎？**  
A: 下載後的效果更完整。Gradio 的 HTML 組件對某些特效有限制，建議下載後在瀏覽器中查看完整效果。

**Q: 可以自定義樣式嗎？**  
A: 可以！下載後編輯 `index.html` 中的 `<style>` 部分即可。

**Q: 如何添加照片？**  
A: 在 `index.html` 的 Hero 區域添加：
```html
<img src="your-photo.jpg" alt="Photo" style="border-radius: 50%; width: 150px;">
```

**Q: 中英文切換不顯示英文內容？**  
A: 當前版本自動從簡歷提取數據。如需完整雙語，需要在 JSON 數據中手動添加英文翻譯。

**Q: 部署後打開很慢？**  
A: 所有資源都內聯在 HTML 中，首次加載可能較慢。可以考慮將CSS和JS提取到單獨文件。

## 💡 高級定制

### 修改配色
在 CSS 中修改 `:root` 變量：
```css
:root {
    --primary: #667eea;    /* 主色 */
    --secondary: #764ba2;  /* 次色 */
    --accent: #00d4ff;     /* 強調色 */
    --dark: #0f172a;       /* 背景深色 */
}
```

### 添加新部分
複製現有的 `<section>` 結構並修改內容。

### 關閉特效
如果覺得特效太多，可以：
- 刪除 `<canvas>` 元素（粒子背景）
- 刪除 `.ripple` 相關代碼（波紋效果）
- 移除動畫 CSS
