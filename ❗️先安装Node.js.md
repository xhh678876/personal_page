# ⚠️ 运行前必读 - Node.js 安装指南

## 问题说明

如果你看到这个文件，说明系统提示"找不到 npm 命令"。

这是因为 **Node.js 还没有安装**，或者没有正确添加到系统环境变量。

---

## 🚀 解决方案

### 步骤 1: 下载 Node.js

访问官网下载：**https://nodejs.org/**

选择：
- **左边的 LTS 版本**（推荐，更稳定）
- 例如：20.x.x LTS

### 步骤 2: 安装 Node.js

1. 双击下载的安装程序
2. **重要**：安装时勾选这些选项：
   - ✅ **Automatically install the necessary tools**
   - ✅ **Add to PATH** （添加到环境变量）
3. 一路点击 "Next" 完成安装

### 步骤 3: 验证安装

安装完成后：

1. **关闭所有命令行窗口**
2. **重启电脑**（重要！确保环境变量生效）
3. 按 `Win + R`，输入 `cmd`，回车
4. 在命令行输入：

```bash
node -v
```

应该显示类似：`v20.11.0`

然后输入：
```bash
npm -v
```

应该显示类似：`10.2.4`

### 步骤 4: 运行项目

验证成功后，双击以下任一文件：

**推荐新手：**
- `启动.bat` - 浏览器网页版

**桌面应用：**
- `运行桌面版.bat` - 独立窗口

**打包分发：**
- `打包成软件.bat` - 生成 .exe（需要先完成上面步骤）

## 🔧 常见问题

### Q: 安装后还是提示找不到 npm？
**A**: 需要重启电脑！环境变量需要重启才能生效。

### Q: 我用的是 Anaconda/Python 环境？
**A**: 在命令行输入 `conda deactivate` 退出 conda 环境，然后重试。

### Q: 已经安装过 Node.js？
**A**: 
1. 检查版本是否太旧（需要 v18 以上）
2. 卸载后重新安装
3. 确保勾选"Add to PATH"

### Q: 公司电脑，没有管理员权限？
**A**: 
1. 下载 Node.js 便携版
2. 或者联系IT部门帮忙安装

## ✅ 安装完成后

1. 重启电脑
2. 双击 `启动.bat`
3. 等待浏览器自动打开
4. 开始使用！

## 💡 推荐配置

### 设置国内镜像（可选，加快下载速度）

安装完 Node.js 后，打开命令行，输入：

```bash
npm config set registry https://registry.npmmirror.com
```

这样后续安装依赖会更快。
